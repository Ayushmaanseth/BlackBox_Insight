Welcome to CareAI Transparency Interface

The aim of the project is to be able to interpret and explain in lay terms the various black-box
algorithms and artificially intelligent models used in different streams, especially in healthcare
where the slightest of miscalculation can cause catastrophe.

The tasks are to explain why a decision was taken, what could have happened if the inputs were
something else, how close were the inputs from the decision being flipped and how can we
audit the model to make it suitable for the real-world data.

LIME (Local Interpretable Model-Agnostic Explanations) can recieve the machine learning result and visulize the prediction of any specific sample.

TensorBoard & What-IF tool can explore datapoint changes and makes it easier to compare multiple models, arrange datapoints by similarity, test algorithmic fairness and compare counterfacts.

We developed an interactive web app which allows ingestion of any model and a dataset against
which the model is trained to provide insights about the model, explaining how any decision being
made is made and what can cause the decision to change.

To start the evaluation and visualization, click "Feature Importance" or "What-IF counterfactual"
button on the top-right.

And then, update the dataset you want to analyze. Input the full name of Labels, Target label and
Zero value.

Finally, click "Submit From" to start the analyzation. Your dataset and analyzation result will be
securely stored in our server and allow you to download.
